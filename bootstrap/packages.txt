# Packages for this Hyprland + Quickshell(ii) setup (Arch).
# Format: one package per line; comments on their OWN line (never inline).
# Lines starting `aur:` need an AUR helper (yay). install.sh parses this.
#
# SELF-CONTAINED: we own the CONFIG (vendored in config/). The desktop's deps come
# from the `mainstream-*` AUR meta-packages below — these pull the full
# qt6 / fonts / python / portal stack transitively. No separate
# "run the end-4 installer" step is needed; `install.sh` installs everything here.
# (The ii *config* = our repo; `quickshell-git` = the Quickshell *runtime* that
# renders it — different things. NOTE: nothing pulls quickshell transitively, so it
# MUST stay listed explicitly or the bar/desktop won't start.)
#
# Upstream (end-4) renamed these from `illogical-impulse-*` to `mainstream-*`; the
# old names no longer exist in the AUR. `illogical-impulse-quickshell-git` was
# replaced by the standalone `quickshell-git`, `illogical-impulse-kde` was dropped
# (its essentials now live in mainstream-basic's deps), and the bibata cursor moved
# to the upstream `bibata-cursor-theme-bin`.

# --- mainstream meta-packages (the base; pull most deps transitively) ---
aur:mainstream-basic
aur:mainstream-hyprland
aur:mainstream-toolkit
aur:mainstream-widgets
aur:mainstream-python
aur:mainstream-audio
aur:mainstream-backlight
aur:mainstream-portal
aur:mainstream-screencapture
aur:mainstream-fonts-themes
# NOTE: mainstream-microtex-git (LaTeX/math rendering in the shell) moved to
# optional-packages.txt — it's a -git source build that can fail/conflict, and the
# desktop runs fine without it. Install it via the optional "latex" group if wanted.

# --- Quickshell runtime (renders the ii config; not pulled transitively) ---
aur:quickshell-git

# --- cursor theme (was illogical-impulse-bibata-modern-classic-bin) ---
aur:bibata-cursor-theme-bin

# --- theming ---
aur:matugen-bin
# Material You color generation hook (called on every wallpaper/color change)
aur:kde-material-you-colors
# GTK theme. The Arch package was renamed adw-gtk3 -> adw-gtk-theme (extra), with no
# provides alias — so the old name errors with "target not found". The theme NAME it
# ships (used by switchwall.sh: gsettings gtk-theme adw-gtk3 / adw-gtk3-dark) is unchanged.
adw-gtk-theme

# --- screenshots: region selection (also used by the optional screen recorder) ---
slurp

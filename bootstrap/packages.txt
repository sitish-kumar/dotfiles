# Packages for this Hyprland + Quickshell(ii) setup (Arch).
# Format: one package per line; comments on their OWN line (never inline).
# Lines starting `aur:` need an AUR helper (yay). install.sh parses this.
#
# SELF-CONTAINED: we own the CONFIG (vendored in config/). The desktop's deps come
# from the `mainstream-*` AUR meta-packages below — these pull the full
# qt6 / fonts / python / portal stack transitively. No separate
# "run the end-4 installer" step is needed; `install.sh` installs everything here.
# (The ii *config* = our repo; `quickshell-git` = the Quickshell *runtime* that
# renders it — different things. NOTE: nothing pulls quickshell transitively, so it
# MUST stay listed explicitly or the bar/desktop won't start.)
#
# Upstream (end-4) renamed these from `illogical-impulse-*` to `mainstream-*`; the
# old names no longer exist in the AUR. `illogical-impulse-quickshell-git` was
# replaced by the standalone `quickshell-git`, `illogical-impulse-kde` was dropped
# (its essentials now live in mainstream-basic's deps), and the bibata cursor moved
# to the upstream `bibata-cursor-theme-bin`.
